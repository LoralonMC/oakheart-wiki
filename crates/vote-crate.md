# Vote Crate

The Vote Crate is your main free reward source! Earn keys by voting daily and open them for resources, emeralds, and rare items.

---

## Getting Vote Keys

Vote on all 5 voting sites daily using `/vote`. Each vote gives you **1 Vote Key**.

- **5 votes/day = 5 Vote Keys/day**
- **35 Vote Keys/week**
- **~150 Vote Keys/month**

## Tips

1. **Vote daily** — Consistency adds up to great rewards over time
2. **Save crystals** — Crate Crystals from rare drops can be traded at Crystal Trader
3. **Check all sites** — Make sure you vote on all 5 for maximum keys
4. **Don't skip days** — You can't make up missed votes
