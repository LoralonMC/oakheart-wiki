# Playing on Bedrock

Oakheart welcomes Bedrock players! The server runs Geyser, so Java and Bedrock players share the same world, chat, and economy.

---

## How to Connect

In your Minecraft Bedrock client, add a new server:

| Field | Value |
|-------|-------|
| Server Name | Oakheart |
| Server Address | `play.oakheart.net` |
| Port | `19132` |

That's it — connect and you'll join the same world as Java players.

---

## Your Bedrock Name

Bedrock accounts appear in chat with a `.` prefix (e.g. `.YourName`) to distinguish them from Java accounts. Your friends will recognize you the same way; the prefix doesn't affect anything else.

---

## What to Expect

Most of Oakheart works the same on Bedrock as on Java — claiming, voting, shops, ranks, crates, and chat all function identically.

A few things behave a little differently on Bedrock:

- **Custom textures** are delivered via a Bedrock resource pack on join. The first download may take a moment.
- **Clickable links in books and chat** — Bedrock supports most of these, but some advanced hover and click interactions are Java-only.
- **Some cosmetics** (hats over armor, certain pet animations) are designed for Java rendering and may look slightly different on Bedrock.

If something doesn't work the way you expect, it's not you — it's a Java/Bedrock parity quirk. Drop a note in Discord and we'll figure it out together.
