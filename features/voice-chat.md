# Voice Chat

Oakheart supports **Simple Voice Chat**, an optional proximity-based voice chat mod. The closer you are to someone, the louder you hear them.

---

## Installation

**Download:** [Simple Voice Chat on Modrinth](https://modrinth.com/plugin/simple-voice-chat)

Supports Fabric, Forge, NeoForge, and Quilt. Place the `.jar` in your `mods` folder and connect to Oakheart — voice chat activates automatically.

{% hint style="info" %}
**No mod?** You can still play without it. Text chat works normally.
{% endhint %}

---

## Controls

| Key | Action |
|-----|--------|
| `V` | Push-to-Talk (hold to speak) |
| `G` | Open voice chat settings |
| `M` | Mute microphone |

Rebind keys in `Options` → `Controls` → `Key Binds` → search "Voice Chat".

---

## Groups

Create private voice groups to talk with specific players regardless of distance:

1. Press `G` to open voice chat menu
2. Click "Groups"
3. Create or join a group

Groups have no distance limit and are private to members.
