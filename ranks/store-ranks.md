# Store Ranks

Support the server and get instant access to perks with Store Ranks! Each rank gives you the same benefits as its equivalent Vote Rank, plus exclusive bonuses and a unique pet companion.

---

## Store Rank Tiers

| Rank | Price | Equivalent Vote Rank | Pet |
|------|-------|---------------------|-----|
| **Hedgehog** | $10 | Rank 5 (Bud) | Hedgehog companion |
| **Squirrel** | $25 | Rank 10 (Root) | Squirrel companion |
| **Otter** | $50 | Rank 15 (Willow) | Otter companion |
| **Owl** | $100 | Rank 20 (Yggdrasil) | Owl companion |
| **Phoenix** | Subscription | Requires Owl (cosmetic only) | Phoenix companion |

{% hint style="info" %}
**Cumulative pricing** — If you own Hedgehog ($10) and want Squirrel, you only pay $15 more (not the full $25).
{% endhint %}

## All Commands by Rank

### Hedgehog ($10)

All commands from Vote Ranks 1-5:
- `/cartographytable` — Open cartography table anywhere
- `/condense` — Convert items to blocks
- `/lay` — Lay down
- `/sit` — Sit on stairs/slabs

**Plus Hedgehog Exclusives:**
- **Keep Inventory** — Keep items on death
- **Instant teleports** — No warmup on `/tpa`, `/home`
- **Hedgehog pet** — Cute companion follows you around

---

### Squirrel ($25)

Everything from Hedgehog, plus Vote Ranks 6-10 commands:
- `/feed` — Restore hunger
- `/loom` — Open loom anywhere
- `/heads` — Decorative head menu
- `/craft` — Open crafting table anywhere
- `/near` — See nearby players
- `/armorstand` — Armor stand editor
- `/dispose` — Trash items quickly
- **Home icons** — Customize icons in `/homes` GUI
- **Squirrel pet** — Fluffy companion follows you around

---

### Otter ($50)

Everything from Squirrel, plus Vote Ranks 11-15 commands:
- `/bellyflop` — Fun emote
- `/stonecutter` — Open stonecutter anywhere
- `/jump` — Teleport up to 30 blocks where you're looking
- `/grindstone` — Open grindstone anywhere
- `/crawl` — Fun emote
- **Otter pet** — Playful companion follows you around

---

### Owl ($100)

Everything from Otter, plus Vote Ranks 16-20 commands:
- `/anvil` — Open anvil anywhere
- `/ptime` — Set personal time
- `/smithingtable` — Open smithing table anywhere
- `/enderchest` — Access ender chest remotely
- `/heal` — Full health restore
- **Owl pet** — Wise companion follows you around

---

### Phoenix (Subscription)

{% hint style="info" %}
**Phoenix** requires owning the Owl rank first. It is a purely cosmetic subscription rank with no gameplay advantages over Owl.
{% endhint %}

Everything from Owl, plus:
- **Exclusive chat tag** — Show your dedication in chat
- **Phoenix pet** — Legendary companion follows you around
- **Future cosmetic additions** — More cosmetics to come!

## Store Rank Exclusives

These perks are **only** available through Store Ranks:

| Perk                | Rank Required |
| ------------------- | ------------- |
| Keep Inventory      | Hedgehog+     |
| Instant teleports   | Hedgehog+     |
| Exclusive pets      | All ranks     |
| Exclusive chat tag  | Phoenix       |

{% hint style="warning" %}
Vote Ranks provide the same core progression (homes, warps, shops, commands) but these exclusives are Store-only perks.
{% endhint %}

## Why Buy Store Ranks?

- **Instant access** — No waiting 200 days to reach max perks
- **Exclusive perks** — Keep Inventory, instant teleports, and more
- **Exclusive pets** — Cute companions unique to each rank
- **Support the server** — Help keep Oakheart running
- **Still vote!** — You can keep voting for tokens, keys, and prestige progress

## Upgrading

Already own a Store Rank? Upgrades only cost the difference:

| Current → Target | Cost |
|------------------|------|
| Hedgehog → Squirrel | $15 |
| Hedgehog → Otter | $40 |
| Hedgehog → Owl | $90 |
| Squirrel → Otter | $25 |
| Squirrel → Owl | $75 |
| Otter → Owl | $50 |

{% endhint %}
