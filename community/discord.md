- **Server news** ‚Äî Announcements, events, changelogs, and sneak peeks

---

## Linking Your Minecraft Account

To send messages in `#ingame-chat` (read-only without linking), connect your Discord and Minecraft accounts:

1. In-game, run `/discord link`
2. You'll receive a code
3. DM that code to the Oakheart bot on Discord
4. Done ‚Äî you're linked, and the **Linked Account** role appears in Discord

Linking is also required to submit a staff application.

## Self-Assign Notification Roles

Pick up notification roles via Discord's Channels & Roles onboarding to get pinged about things you care about:

- **`ping-announcements`** ‚Äî Server announcements
- **`ping-events`** ‚Äî Lotteries, contests, build competitions
- **`ping-voting`** ‚Äî Daily vote reminder
- **`ping-changelog`** ‚Äî Patch notes and updates

---

## Reporting Issues

Open a support ticket via the `#tickets` channel and pick the category that fits:

- **In-game Support** ‚Äî gameplay questions, bug reports, stuck-player issues
- **Player Report** ‚Äî griefing, harassment, rule-breaking
- **Punishment Appeal** ‚Äî appeal a ban or mute
- **Billing** ‚Äî purchase or refund issues
- **Bug Report** ‚Äî technical issues with the server or plugins

Staff will respond as soon as we can.
